# Claude Agent SDK Integration Examples

This directory contains practical examples of integrating the Agentic Learning SDK with Claude Agent SDK.

## Files

### basic_integration.py
Simple 3-line integration example showing the core pattern.

### research_agent.py
Complete research agent implementation with memory for tracking research sessions and findings.

### code_review_assistant.py
Code review assistant that remembers feedback patterns and project standards across reviews.

### multi_agent_system.py
Example showing multiple agents sharing memory through a common agent name.

## Usage

Run examples with:
```bash
python basic_integration.py
python research_agent.py
python code_review_assistant.py
python multi_agent_system.py
```

## Requirements

- `anthropic` package
- `agentic-learning-sdk` package
- Valid Anthropic API key
